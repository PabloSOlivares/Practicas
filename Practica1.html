Es un lenguaje de marcado de texto que sirve para presentar paginas web desde el punto de vista del front end; un front end en español significa la parte final que un usuario de internet vera. Una etiquetra marca delimitada cada uno de los elementos que compone un documento html. Las paginas web estan formadas por dos partes, el encabezado y es cuerpo de la pagina, la primera tiene la funcion de mostrar configuraciones y el cuerpo es donde se mostrara en si toda la pagina. La etiqueta html es el inicio de una pagina web en cualquier version pero existe una etiqueta que se llama !Doctype html que indica una pagina web con la version 5
